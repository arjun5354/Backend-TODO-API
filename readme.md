# Smart ToDo API

A RESTful backend API for task management built using **Node.js**, **Express**, and **MongoDB**.

---

## Features
- User authentication using JWT
- Create, read, update, and delete tasks
- Task priority and status management
- Soft delete (archive tasks)
- Pagination and filtering
- Swagger API documentation

---

## Tech Stack
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- Swagger (OpenAPI)
